This folder contains four python files.

1) preprocess.py is used to preprocess the raw data, drop it if it contains more than 1/4 of NaN data.
2) features.py is used to extract features from the raw data, is the same file used in project1.
3) decision_tree.py is used to split data to training set and testing set and then trained and evaluate the model.
4) test.py is an python script that runs everything and used as entry point.

Run "python test.py" and then enter the filename(plus the extension) you want to predict.
